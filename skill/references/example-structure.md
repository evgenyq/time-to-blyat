# Example Dashboard Structure

This is the structural reference for the Time to Blyat™ dashboard. Use as a template — adapt data, quotes, and metrics to the actual analysis results.

## HTML Sections (in order)

### 1. Hero
```html
<div class="hero">
  <div class="hero-badge">Analytics Report • {date_range}</div>
  <h1>Time to<br>Блять™</h1>
  <p class="hero-subtitle">A Self-Study in Developer Rage</p>
  <div class="hero-number">
    <div class="number" id="hero-counter">{median_minutes}</div>
    <span class="unit">MINUTES</span>
    <span class="label">median time from session start to first profanity</span>
  </div>
  <div class="scroll-hint">↓ scroll for the damage report</div>
</div>
```
- Pulsing radial gradient glow behind the title
- Counter animates from 0 to median value on page load
- Subtle parallax on mouse move

### 2. Overview Stats (Section 01)
Grid of 4 stat cards:
- Sessions analyzed (total count)
- Zen mode % (calm sessions)
- Rage mode % (frustrated sessions)
- Carry-over trauma (started already angry)

### 3. Frustration Curve (Section 02)
Horizontal bar chart — "How fast does the rage arrive?"
Buckets: 0–10 min, 10–20, 20–30, 30–45, 45–60, 60–90, "@ start"
- Bars animate width on scroll-into-view
- Tags: "⚡ Speed Run" for fastest, "🐌 Slow Burn" for slowest, "😤 Carry-over" for pre-frustrated
- Color gradient: red (fast) → orange → yellow → blue (slow) → purple (carry-over)

### 4. Rage Scale (Section 03) — "The Richter Scale of Rage"
Table of expressions sorted by intensity:
```
L1 (green):  "блин"           → "Hmm, that's not ideal"
L2 (yellow): "опять"          → "We've been through this"
L3 (orange): "бля / чувак"    → "Are you serious right now?"     [MOST COMMON badge]
L4 (red):    "какого хера"    → "What the actual..."
L5 (pink):   "НУ ПОЧЕМУУУ??" → "🔥 Keyboard in danger 🔥"
```
Each row: count badge + expression + English translation + dot indicators (one dot per occurrence)

### 5. Root Cause Analysis (Section 04) — "What Sets Me Off"
2-column grid of trigger cards:
- Count (big number) + description + proportional bar
- Example categories: build failures, AI ignoring instructions, permission prompts, AI losing context, etc.

### 6. Hall of Fame (Section 05) — "Greatest Hits 💀"
3 quote cards (gold/silver/bronze):
```
🥇 Peak Rage — {date}, {project}
"{actual quote}"
Context: {what was happening} • {fun metadata}

🥈 Existential Crisis — {date}, {project}
"{actual quote}"
Context: {description}

🥉 {Category} — {date}, {project}
"{actual quote}"
Context: {description}
```
Cards have colored left border, hover animation (slide right)

### 7. Calendar Heatmap (Section 06) — "{Month} — Heatmap of Pain"
7-column grid (Mon-Sun), one cell per day:
- `.calm` — no frustrated sessions (dark bg)
- `.hot1` — 1-2 frustrated sessions (dark red bg)
- `.hot2` — 3-4 frustrated sessions (medium red)
- `.hot3` — 5+ frustrated sessions (bright red + glow + 🔥)
- Hover shows tooltip with details
- Future/non-analyzed days at reduced opacity

### 8. By Project (Section 07) — "Where The Pain Lives"
Horizontal bar chart, one bar per project:
- Sorted by frustration count descending
- Colors: red → orange → yellow → blue
- Bars animate on scroll

### 9. Footer — "The Pattern™"
```
Work calmly for {median} min → something breaks → "блин" → 5 min later → "бля" → 5 more min → full мат → repeat.
```
Color-coded escalation stages.

Disclaimer: "Generated by the very AI that caused {rage_pct}% of these sessions. No keyboards were harmed in the making of this report. Probably."

## Design Tokens

```css
--bg: #0a0a0f;        /* main background */
--card: #14141f;       /* card background */
--border: #2a2a3a;     /* subtle borders */
--text: #e0e0e8;       /* primary text */
--text-dim: #8888a0;   /* secondary text */
--red: #ff4444;        /* rage, L4 */
--orange: #ff8c42;     /* frustrated, L3 */
--yellow: #ffd166;     /* annoyed, L2 */
--green: #06d6a0;      /* calm, L1 */
--blue: #4ea8de;       /* slow burn */
--purple: #b388ff;     /* carry-over */
--pink: #ff6b9d;       /* nuclear, L5 */
```

Fonts: JetBrains Mono (data/code), Inter (body text) — loaded via Google Fonts.

## Interactive Elements

- Scroll-reveal animations (IntersectionObserver, threshold 0.15)
- Bar charts animate width with cubic-bezier easing on scroll
- Hero counter animates 0 → value with easeOutCubic over 2 seconds
- Subtle mouse parallax on hero section
- Calendar day hover → scale(1.1) + tooltip
- Quote cards hover → translateX(4px) + border color change
- Film grain overlay (SVG noise filter, opacity 0.03)
